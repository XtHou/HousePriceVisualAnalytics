## first install 

*Mac*

Install `GDAL` on the command line first, e.g., using `homebrew`

```
brew install gdal
```

Then install `rgdal` and `rgeos`

```
install.packages("rgdal", type = "source")
install.packages("rgeos", type = "source")
```

### Install geojsonio

```
install.packages("geojsonio")
library("geojsonio")
```
